# Multi-Channel Usage

Ferrumyx provides seamless integration across multiple communication channels, allowing you to interact with the system wherever you work. This enables flexible workflows that adapt to your research environment.

## Supported Channels

Ferrumyx supports the following communication platforms:

- **Web Interface**: Full-featured web application at your Ferrumyx URL
- **WhatsApp**: Direct messaging with the Ferrumyx bot
- **Slack**: Integration with Slack workspaces and channels
- **Discord**: Bot integration for Discord servers

## Channel Features

### Web Interface
- **Full functionality**: Access to all Ferrumyx features
- **Visual results**: Charts, graphs, and interactive visualizations
- **File uploads**: Drag-and-drop support for data files
- **Session management**: Persistent conversations and project organization

### Chat Interfaces (WhatsApp/Slack/Discord)
- **Quick queries**: Fast access for simple questions
- **Notifications**: Automated alerts for long-running analyses
- **Collaboration**: Share results with team members
- **Mobile access**: Research on-the-go

## Seamless Transitions

Ferrumyx maintains conversation context across channels:

1. **Start on web**: Begin complex analysis in the web interface
2. **Continue on mobile**: Switch to WhatsApp for quick follow-ups
3. **Share with team**: Move discussion to Slack for collaboration
4. **Get alerts**: Receive notifications via Discord

## Best Practices

### Choosing the Right Channel
- **Web**: Complex queries, data visualization, file uploads
- **Chat**: Quick questions, status updates, mobile access
- **Team channels**: Collaborative research, result sharing

### Context Management
- Use consistent thread IDs for related research
- Reference previous conversations when switching channels
- Save important results to project folders

### Notification Setup
Configure automated monitoring across channels:
```
Monitor new publications about immunotherapy in renal cell carcinoma
```
- Set up alerts in your preferred channel
- Choose notification frequency (daily/weekly)
- Specify sources (PubMed, clinicaltrials.gov, etc.)
