# Sample Files Directory

This directory contains sample data files for testing and development.

## Files

- sample.fasta - Sample FASTA sequence file for bioinformatics testing
- sample.fastq - Sample FASTQ sequence file for quality control testing
- ferrumyx_colab.ipynb - Jupyter notebook for Google Colab integration

## Usage

These files are used by the testing framework and can be used for:

- Validating bioinformatics pipeline functionality
- Testing sequence analysis workflows
- Demonstrating Colab integration capabilities

See docs/user/research-workflows.md for usage examples.
